### **Features:**

- Anti-aliasing mode selection (Off, FXAA, SMAA, TAA)
- TAA sharpening adjustment
- Shadow distance control
- LOD bias adjustment
- VSync toggle
- Post-processing controls:
- Bloom intensity
- Vignette intensity
- Motion blur
- Chromatic aberration
- Contrast
- Saturation
